Triple Fault OS

Description

This is an operating system coded in Rust. I'm following Philipp Oppermann's excellent blog 'Writing an OS in Rust' (linked below) to teach myself Rust while creating an interesting project. The OS contains a basic kernel with VGA text output, exceptions and interrupts, paging, a heap allocator, and the beginnings of asynchronous multitasking.
